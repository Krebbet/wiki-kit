# Trends in the industry

- **Coding-agent evaluation moved past "fix one GitHub issue."** Four benchmarks landed in one week — SWE-Cycle (full Env→Impl→TestGen→FullCycle cycle), SWE-Atlas (Scale: Codebase Q&A + Test Writing + Refactoring), SWE-Chain (release-level package-upgrade chains), SWE-WebDevBench (vibe-coding platforms). SWE-bench-Verified saturation is forcing the eval target onto the *whole software lifecycle*; reported FullCycle solve rates collapse to ~<14%, exposing how much friction pre-configured benchmark environments were hiding.
- **Benchmark contamination became a first-order dispute.** SWE-Cycle reports ~35% exact 5-gram match on reference patches and models locating buggy files with ~76% accuracy *without repo access* — a direct challenge to SWE-bench Verified/Pro headline numbers. *(synthesis: absolute solve-rate comparisons across these benchmarks are no longer safely comparable until contamination is filtered uniformly.)*
- **Agent memory is being re-validated against text-search baselines, and losing.** GroupMemBench (first multi-party memory benchmark) reports a bare BM25 baseline (~43%) strictly Pareto-dominating four of five extraction-based memory systems; best system only ~46%; Mem0 collapses to ~4.67% on knowledge-update. Paired with a new Storage→Reflection→Experience survey, the field is converging on "current extraction-based memory doesn't survive the multi-party/production regime."
- **Self-improvement is consolidating on RL-trained-curation / distillation over a frozen executor.** SDAR (on-policy self-distillation + token-level gating; skills used at training, internalized into weights, not needed at inference) joins SkillOS and skill-distillation. *(synthesis: the separate-curator / internalize-then-discard pattern the wiki's roles/skills proposal synthesized is now multiply instantiated across independent groups.)*
- **Context engineering now has a primary-vendor canonical reference.** Anthropic's "Effective Context Engineering for AI Agents" (attention-budget / "context rot"; the JIT-retrieval + compaction + structured-note-taking triad) spread through practitioner channels and LangChain Interrupt 2026 — context engineering is being treated as a discipline distinct from prompt engineering, with Claude Code's grep/glob JIT retrieval named as a production instance.
