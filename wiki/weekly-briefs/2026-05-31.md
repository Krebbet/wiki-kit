# Trends in the industry
- **Harness-vs-model scaling debate crystallizing.** Multiple independent signals this week — a position paper (arXiv 2605.26112) coining "harness scaling" as the next frontier after model scaling; AAIF institutionalizing AGENTS.md and MCP as open standards; Latent Space's async-agents episode describing harness-in/out-of-box as the core architectural decision. The thesis that the execution layer around the model is now the primary design object is moving from research framing to industry consensus.
- **Async/long-running agents entering production.** Cognition's Latent Space interview describes a three-wave shift (copilot → local agents → async cloud agents) triggered by the Dec 2025 model inflection (Opus 4.5/GPT 5.2). Devin metrics: 16%→80% commit share, 7x merged PR growth (collect-but-confirm). Repo setup named as the hardest recurring operational problem; always-on agents as "permanent product owners" as the direction.
- **Open standardization of agent interoperability.** OpenAI + Anthropic + Block co-founded the Agentic AI Foundation (AAIF) under the Linux Foundation, donating AGENTS.md, MCP, and Goose as open standards with neutral stewardship. 180+ member orgs including Google, Microsoft, AWS (vendor-stated). Marks the transition from proprietary agent formats to an open ecosystem — a governance inflection.
- **Coding agents hitting commercial inflection.** Willison's May 27 post: Anthropic's enterprise pricing shifted from flat seats to $20/seat + API usage (Nov 2025, disclosed Apr 2026); April 2026 is the named revenue inflection. SpaceX S-1 cites $1.25B/month in AI compute. Enterprise customers discovering unexpectedly large bills from coding-agent usage. Anthropic reportedly approaching first profitable quarter.
- **Agent security widening from detection to theory.** arXiv 2605.17634 argues prompt injection defense via data-instruction separation is insufficient even in principle — contextual manipulation attacks evade it by design, via Contextual Integrity theory. This is a theoretical complement to last week's ADR detection paper (which catches downstream malicious *actions* but not the injection itself). The attack surface is larger than current defenses assume.
