OpenAI filed its confidential IPO S-1 the same week (May 22) at an $852B–$1T target, disclosing $25B annualized revenue but $1.22 loss per $1 revenue in Q1. The "AI labs are capital sinks" narrative is cracking: one is profitable, one is going public, both this week.

- **Enterprise AI distribution is consolidating through Big Four + PE** — not sales teams. Anthropic's KPMG alliance (276K employees, Digital Gateway embed) and its $1.5B JV with Blackstone/Goldman/HF (forward-deployed Anthropic engineers inside PE portfolio companies) are structurally similar to OpenAI's DeployCo ($4B, launched the same week as the JV). The delivery model is embedding engineers, not selling seats. This is a services-layer land-grab, not a licensing play.

- **Hyperscaler "managed agent execution" APIs converged within three weeks.** Google (Managed Agents API, I/O May 19), Amazon (AgentCore, prior), Microsoft (Copilot Studio computer-use GA, May 13) all shipped "run agents in a secure, hosted environment" primitives. The competitive axis has shifted from model quality to runtime isolation guarantee — which vendor's sandbox you trust with agentic workloads.

- **AI developer toolchains are now confirmed active attack targets.** TanStack "Mini Shai-Hulud" (May 11, TeamPCP) compromised 170+ npm + 2 PyPI packages via GitHub Actions cache poisoning, hitting two OpenAI employee devices and one Mistral developer device. The attack produced the first malicious npm package with valid SLSA provenance — directly invalidating "require attestation" as a sufficient supply-chain control.

- **AI regulatory vacuum deepened.** White House pulled its voluntary 90-day pre-launch review EO hours before signing (May 21) — Trump cited competitiveness concerns. EU AI Act Omnibus already delayed to 2027–2028. Enterprise AI governance remains effectively self-regulated for the foreseeable future; platform vendors (Microsoft Purview, Salesforce Agent Fabric, Google Model Armor) fill the vacuum.
