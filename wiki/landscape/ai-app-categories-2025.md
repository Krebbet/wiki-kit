# AI Application Categories 2025

Enterprise application-layer spend broken down by category — departmental, vertical, horizontal — with dollar figures from Menlo's 2025 report. Scaffold for per-offering pages. Numbers are point-in-time (end of 2025); re-verify before quoting.

## Headline

- Application layer reached **$19B of $37B total enterprise AI spend** in 2025 — **>50% of spend for the first time** (Menlo 2025-12-09).
- **Startups captured 63% of application-layer revenue**, up from 36% in 2024; incumbents hold ~56% at the infrastructure layer (Menlo 2025-12-09).
- Startup share by function: **product + engineering 71%, sales 78%, finance + operations 91%**; incumbents retain IT and data science on reliability / integration depth (Menlo 2025-12-09).

### Coding specifics (2025-12-09)

- Market jumped from **$550M (2024) → $4.0B (2025)** — attributed to models interpreting entire codebases and executing multi-step tasks.
- **50% of developers use AI coding tools daily**; 65% in top-quartile orgs. Reported **15%+ velocity gains** across SDLC.
- Code completion: **$2.3B**. Code agents and AI app builders grew from near-zero.
- Named offerings across SDLC: **Lovable** (prototyping), **Open Hands** (refactoring), **Weaver** (design-to-code), **Meticulous** (QA), **Graphite** (PRs), **Resolve** (SRE), **Harness** (deployment).

May 2025 reference point: one high-growth SaaS CTO reported **~90% of code AI-generated** via Cursor and Claude Code, up from 10–15% twelve months earlier with GitHub Copilot — bleeding-edge but a leading indicator (a16z 2025-05).

### Healthcare — ambient clinical scribes (2025-12-09)

- Market size: **$600M in 2025 (2.4x YoY)**.
- Unicorns cited: **Abridge**, **Ambience**.
- Market leader: **Nuance DAX Copilot**.
- Claim: scribes reduce documentation time >50%; clinicians spend ~1 hour documenting per 5 hours of care (Menlo, citing practitioner data).

## Emerging agent platforms (2026-Q1 funding rounds)

Sourced from direct funding announcements, not the Menlo/a16z landscape surveys. Adds to the "agent platforms" sub-category above.

- **[[lio]]** — enterprise procurement agents. $30M Series A (a16z-led), announced 2026-03-05; $33M total raised. YC Spring '23. Positioned as a buy-side replacement for BPO spend and Ariba / Oracle. Founder-quoted metrics only — treat as marketing until corroborated. (TechCrunch 2026-03-05.)
- **[[trace]]** — knowledge-graph + context-engineering orchestration layer on top of email / Slack / Airtable / Jira. $3M seed, announced 2026-02-26. London-based, YC Summer 2025. Positions itself as "the manager that knows where to put the [labs'] brilliant interns." Seed-stage; no customers or deployment evidence in source. (TechCrunch 2026-02-26.)

### Incumbent-platform counter-moves (2026-04)

- **[[salesforce]] — Salesforce Headless 360** (2026-04-15 TDX 2026): entire Salesforce platform exposed as APIs, MCP tools, and CLI commands for agents; 60+ MCP tools, 30+ preconfigured coding skills, Agentforce Experience Layer for native rendering across Slack / Mobile / ChatGPT / Claude / Gemini / Teams, and **Agent Fabric** as a multi-vendor agent control plane. Canonical "incumbent becomes agent infrastructure" pivot.
- **[[c3-ai]] — C3 Code** (GA 2026-04-08): agentic coding tightly coupled to C3 Agentic AI Platform, generating full-stack Enterprise AI from natural-language prompts. Vendor-marketing-heavy; no named C3 Code customers yet.
- **[[google-cloud-agentic-partner-fund-2026-04]]** (2026-04-22): $750M partner fund for agentic-AI deployments via GSIs; Gemini Enterprise agent catalog with Adobe / Atlassian / Oracle / Salesforce / ServiceNow / Workday as named ISV agents — hyperscaler runtime + agent marketplace.

### Frontier LLM releases (2026-04)

- **[[anthropic-claude-family]] — Claude Opus 4.7** (2026-04-16): replaces Opus 4.6 as default. Vendor benchmark lifts include CursorBench 58→70, Rakuten-SWE-Bench 3× production tasks, XBOW visual-acuity 54.5→98.5. Pricing unchanged ($5/M input, $25/M output). New `xhigh` effort level.

## App-leaderboard reality (2026-04)

- **Microsoft 365 Copilot** leads enterprise chat (ChatGPT has closed the gap meaningfully) (a16z 2026-04).
- **GitHub Copilot** still the coding leader for enterprises — *despite* the Claude Code / Codex / Cursor discourse (a16z 2026-04).
- **65% of enterprises prefer incumbents when available**, citing trust, integration, and procurement simplicity (a16z 2026-04).
- Prior a16z data point: Microsoft Copilot NPS dropped **48 points** after the same developers tried Cursor — incumbents set low ceilings that users don't recognize until they try better tools (a16z 2025-05).

## Product-led growth pattern

- **27% of AI application spend flows through PLG motions** (vs 7% for traditional SaaS, ~4x) (Menlo 2025-12-09).
- Including "shadow AI" (personal-card ChatGPT Plus etc., ~27% of usage work-related per NBER WP 34255, Sept 2025), PLG-driven share approaches **~40% of application spend** (Menlo 2025-12-09).
- PLG scalers named: **Cursor** (reported $200M revenue before first enterprise sales rep — precise figure flagged, see [[open-questions-2026-04]] C7), **n8n** (open-source community → contracts), **ElevenLabs**, **Gamma**, **Wispr Flow**, **Lovable**, **OpenRouter**, **fal**.
- May-2025 observation: prosumer pull drives enterprise buys — ChatGPT, ElevenLabs, coding apps entered via employee use; enterprise contracts followed the brand (a16z 2025-05).

## Infrastructure layer — $18B

- Foundation model APIs: **$12.5B**. Model training infra: **$4.0B**. AI infrastructure (storage / retrieval / orchestration): **$1.5B** (Menlo 2025-12-09).
- Inference-platform startups cited: **Fireworks, Baseten, Modal, Together** — claiming "2x+ speedups" via handwritten / fused kernels and managed GPU fleets.
- Observability / tooling: **LangChain, Braintrust, Judgment Labs**.
- AI-native infra with strong growth: **Temporal, Supabase, Neon, Pinecone**.
- **Databricks, Snowflake, MongoDB, Datadog "re-accelerating"** as even AI-native app builders default to existing platforms (Menlo 2025-12-09). See [[databricks]], [[snowflake]].

## Techniques-under-the-hood prevalence (2025-12-09)

Descending prevalence of customization technique in enterprise deployments:

**prompt design → RAG → fine-tuning → tool calling → context engineering → RL**

RL / post-training is "niche, used primarily by frontier teams" (Menlo 2025-12-09).
